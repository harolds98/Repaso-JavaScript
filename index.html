DOOM

El Document Object Model, al que también se conoce como DOM por sus siglas en inglés, es un concepto que nos permite manipular los elementos de un documento HTML directamente desde nuestro código JavaScript. Este concepto se basa en la idea de nodos que se conectan para formar un árbol jerárquico.
